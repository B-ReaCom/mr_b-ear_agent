# Mr. B-EAR - ALVO 1.9 AIエージェント

ReaCom ALVO 1.9製品に関する質問に回答するAIエージェント。

## デプロイ方法

このプロジェクトはVercelで自動デプロイされます。

## ファイル構成

- `index.html` - メインアプリケーション（スタンドアロン版）
- `README.md` - このファイル

## 更新方法

1. `index.html`を編集
2. GitHubにコミット＆プッシュ
3. Vercelが自動的にデプロイ（約30秒）
